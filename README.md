# Storefront – E-Commerce Store 🛍️

## 📖Project Description

Storefront is a fully responsive E-Commerce Store built using HTML, CSS, and Vanilla JavaScript. The application fetches real-time product data from the Fake Store API and allows users to search, filter, sort, view product details, manage a shopping cart, save wishlist items, and enjoy a smooth shopping experience across all devices.

## ✨Features

- Fetch products from the Fake Store API
- Skeleton loading screen
- Error handling with Retry button
- Responsive product grid
- Product count display
- Real-time product search
- Dynamic category filtering
- Product sorting
- Clear all filters
- Product detail modal
- Quantity selector
- Shopping cart with dynamic subtotal
- Checkout confirmation modal
- Cart data saved using LocalStorage
- Dark / Light mode with LocalStorage
- Responsive design for desktop, tablet, and mobile
- Mobile hamburger menu
- Full-screen cart drawer on mobile
- Wishlist (Bonus)
- Load More Pagination (Bonus)
- Debounced Search using Closures (Bonus)
- Smooth animations and transitions (Bonus)
- Product Comparison (Bonus)

## 🚀How to Run Locally

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension (if needed).
4. Right-click on **index.html**.
5. Select **Open with Live Server**.
6. The project will open in your browser.

## 📚What I Learned

This project helped me improve my frontend development skills by building a complete E-Commerce Store using Vanilla JavaScript. I learned how to work with REST APIs, asynchronous programming using async/await, DOM manipulation, responsive web design, and LocalStorage. One of the biggest challenges was making search, filtering, sorting, wishlist, pagination, and the shopping cart work together without affecting each other. I also improved my debugging skills by using browser developer tools to identify and fix JavaScript errors, file path issues, and responsive layout problems.
